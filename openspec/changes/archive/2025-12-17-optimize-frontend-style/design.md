# Design Guidelines

## Aesthetic
- **Modern**: Clean lines, subtle gradients, potentially slight transparency (glassmorphism).
- **Concise**: Minimalist approach, avoiding unnecessary decorative elements.
- **Dark Theme**: Consistent with `main.css` color palette.

## Key Changes
- **Navigation Bar**:
    -   Use `backdrop-filter: blur()` for a modern feel.
    -   Cleaner border colors using variables.
- **Typography**:
    -   Leverage `Inter` and `JetBrains Mono` from `main.css`.
- **Interactions**:
    -   Subtle transitions for hover states.

## Constraints
- No new libraries (e.g., no Tailwind if not already present, no new npm packages).
- No functional changes.
