## ADDED Requirements

### Requirement: Home Page Dashboard Layout
The home page MUST present a concise, modern dashboard layout that groups functional areas logically.

#### Scenario: Dashboard Section Grouping
- **WHEN** a user visits the home page
- **THEN** they should see statistics at the top
- **AND** the workflow guide and quick actions should be grouped together for better workspace focus
- **AND** the recent activity should be clearly visible next to the workspace area

### Requirement: Modern Visual Aesthetics
The home page and its components MUST use a refined modern aesthetic including subtle gradients, glassmorphism, and polished transitions.

#### Scenario: Visual Polish
- **WHEN** a user interacts with the dashboard
- **THEN** background should feature sophisticated radial gradients
- **AND** cards should have a subtle backdrop blur (glassmorphism)
- **AND** transitions should be smooth and responsive
